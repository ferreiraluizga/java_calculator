# Calculadora
Calculadora feita em Java com uso do JFrame

## Funcionamento
- IDE usada para desenvolvimento: Apache Net Beans 21
- Baixe o projeto e abra na sua IDE
- Execute o arquivo Calculadora.java (src/calculadora/Calculadora.java)
